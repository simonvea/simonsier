# Task Completion Checklist

After completing a task:

1. **Build check:** Run `npm run build` to ensure the site builds without errors
2. **Dev preview:** Run `npm run serve` to visually verify changes in the browser
3. **No test suite** — there are no automated tests
4. **No linter** — no ESLint/Prettier configured; review manually
5. **Content language:** Ensure any new content is in Norwegian unless explicitly asked otherwise
6. **Production build:** For deploy-affecting changes, verify with `npm run build:prod`
